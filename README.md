# School Project 1 aus TLS

Eine Sammlung von Java-Übungsprogrammen aus dem Schulunterricht zum Erlernen der Java-Programmierung.

## Projektstruktur

### _1_Einstieg - Grundlagen
Grundlegende Java-Konzepte und erste Programme:
- **[HelloWorld.java](src/_1_Einstieg/HelloWorld.java)** - Das klassische "Hello World" Programm
- **[Division.java](src/_1_Einstieg/Division.java)** - Division mit Ganzzahlen und Restberechnung
- **[Variable.java](src/_1_Einstieg/Variable.java)** - Umgang mit Variablen und Datentypen
- **[Rechteck.java](src/_1_Einstieg/Rechteck.java)** - Berechnung von Fläche und Umfang eines Rechtecks
- **[Umrechnen.java](src/_1_Einstieg/Umrechnen.java)** - Einheitenumrechnungen
- **[Mitarbeiterdaten.java](src/_1_Einstieg/Mitarbeiterdaten.java)** - Verarbeitung von Mitarbeiterdaten
- **[MeinName.java](src/_1_Einstieg/MeinName.java)** - Einfache Textausgabe
- **[Willkommen.java](src/_1_Einstieg/Willkommen.java)** - Begrüßungsprogramm

**Fokus:** Scanner-Nutzung, Datentypen, mathematische Operationen, Ein-/Ausgabe

### _2_Verzweigungen - Kontrollstrukturen
Programme mit Verzweigungen und bedingter Logik:
- **[PinCode.java](src/_2_Verzweigungen/PinCode.java)** - PIN-Überprüfung mit if-Bedingungen
- **[Rabattrechner.java](src/_2_Verzweigungen/Rabattrechner.java)** - Rabattberechnung nach Bestellwert
- **[Rabattrechner_Methoden.java](src/_2_Verzweigungen/Rabattrechner_Methoden.java)** - Rabattrechner mit Methoden
- **[Versandkostenrechner.java](src/_2_Verzweigungen/Versandkostenrechner.java)** - Berechnung von Versandkosten
- **[IsEven.java](src/_2_Verzweigungen/IsEven.java)** - Prüfung auf gerade/ungerade Zahlen
- **[BuildBoolean.java](src/_2_Verzweigungen/BuildBoolean.java)** - Boolean-Logik und Bedingungen
- **[AdvancedArray.java](src/_2_Verzweigungen/AdvancedArray.java)** - Erweiterte Array-Operationen
- **[Bild.java](src/_2_Verzweigungen/Bild.java)** - Bildverarbeitung
- **[VerbrauchBenzin.java](src/_2_Verzweigungen/VerbrauchBenzin.java)** - Benzinverbrauchsberechnung

**Fokus:** if-else Verzweigungen, Boolean-Logik, Methoden

### _3_UML - UML-Diagramme
Visualisierung von Programmstrukturen und Abläufen:
- **[Online-Bestellung.drawio](src/_3_UML/Online-Bestellung.drawio)** - UML-Diagramm für Online-Bestellungen
- **[UserLogin.drawio](src/_3_UML/UserLogin.drawio)** - UML-Diagramm für User-Login-Prozesse

**Fokus:** UML-Klassendiagramme, Anwendungsfälle, Visualisierung von Abläufen

### _4_Schleifen - Schleifen und Wiederholungen
Programme mit for- und while-Schleifen:
- **[DurchschnittsTemperatur.java](src/_4_Schleifen/DurchschnittsTemperatur.java)** - Durchschnittsberechnung von Temperaturen
- **[StromanbieterTestKlassenarbeit.java](src/_4_Schleifen/StromanbieterTestKlassenarbeit.java)** - Testprogramm zu Stromanbietern
- **[Weihnachtsbaum.java](src/_4_Schleifen/Weihnachtsbaum.java)** - Ausgabe eines Weihnachtsbaums mit Schleifen
- **[WiederholtAusgeben1_100.java](src/_4_Schleifen/WiederholtAusgeben1_100.java)** - Zahlen von 1 bis 100 ausgeben
- **[WiederholtAusgebenName.java](src/_4_Schleifen/WiederholtAusgebenName.java)** - Namen mehrfach ausgeben
- **[WiederholtAusgebenNFakultaet.java](src/_4_Schleifen/WiederholtAusgebenNFakultaet.java)** - Fakultät mit Schleifen berechnen
- **[WiederholtAusgebenZahl.java](src/_4_Schleifen/WiederholtAusgebenZahl.java)** - Zahl mehrfach ausgeben
- **[DurchschnittsTemperatur.drawio](src/_4_Schleifen/DurchschnittsTemperatur.drawio)** - Diagramm zur Temperaturberechnung

**Fokus:** for- und while-Schleifen, Zähl- und Abbruchbedingungen, einfache Algorithmen

### _OutOfContext - Erweiterte Programme
Zusätzliche Programme und erweiterte Konzepte:
- **[AdvancedArray.java](src/_OutOfContext/AdvancedArray.java)** - Erweiterte Array-Operationen mit ArrayList
- **[BuildBoolean.java](src/_OutOfContext/BuildBoolean.java)** - Objektorientierte Programmierung mit Klassen und Boolean-Logik

**Fokus:** ArrayList, Objektorientierung, Switch-Case, Packages

## Zweck
Mitschriften und praktische Übungen zum Erlernen der Java-Programmiergrundlagen im Schulkontext.

## Voraussetzungen

- Java Development Kit (JDK) 17 oder höher
- Eine Java-IDE wie IntelliJ IDEA oder VS Code

## Ausführen der Programme

1. Repository klonen oder herunterladen.
2. In der IDE das gewünschte Java-File im `src`-Verzeichnis öffnen.
3. Das Programm ausführen (meistens Rechtsklick auf die Datei → `Run`).

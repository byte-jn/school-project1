# School Project

Eine Sammlung von Java-Übungsprogrammen aus dem Schulunterricht zum Erlernen der Java-Programmierung.

## Projektstruktur

### Einstieg_1 - Grundlagen
Grundlegende Java-Konzepte und erste Programme:
- **[HelloWorld.java](src/Einstieg_1/HelloWorld.java)** - Das klassische "Hello World" Programm
- **[Division.java](src/Einstieg_1/Division.java)** - Division mit Ganzzahlen und Restberechnung
- **[Variable.java](src/Einstieg_1/Variable.java)** - Umgang mit Variablen und Datentypen
- **[Rechteck.java](src/Einstieg_1/Rechteck.java)** - Berechnung von Fläche und Umfang eines Rechtecks
- **[Umrechnen.java](src/Einstieg_1/Umrechnen.java)** - Einheitenumrechnungen
- **[Mitarbeiterdaten.java](src/Einstieg_1/Mitarbeiterdaten.java)** - Verarbeitung von Mitarbeiterdaten
- **[MeinName.java](src/Einstieg_1/MeinName.java)** - Einfache Textausgabe
- **[Willkommen.java](src/Einstieg_1/Willkommen.java)** - Begrüßungsprogramm

**Fokus:** Scanner-Nutzung, Datentypen, mathematische Operationen, Ein-/Ausgabe

### Verzweigungen_2 - Kontrollstrukturen
Programme mit Verzweigungen und bedingter Logik:
- **[PinCode.java](src/Verzweigungen_2/PinCode.java)** - PIN-Überprüfung mit if-Bedingungen
- **[Rabattrechner.java](src/Verzweigungen_2/Rabattrechner.java)** - Rabattberechnung nach Bestellwert
- **[Rabattrechner_Methoden.java](src/Verzweigungen_2/Rabattrechner_Methoden.java)** - Rabattrechner mit Methoden
- **[Versandkostenrechner.java](src/Verzweigungen_2/Versandkostenrechner.java)** - Berechnung von Versandkosten
- **[IsEven.java](src/Verzweigungen_2/IsEven.java)** - Prüfung auf gerade/ungerade Zahlen
- **[BuildBoolean.java](src/Verzweigungen_2/BuildBoolean.java)** - Boolean-Logik und Bedingungen
- **[AdvancedArray.java](src/Verzweigungen_2/AdvancedArray.java)** - Erweiterte Array-Operationen
- **[Bild.java](src/Verzweigungen_2/Bild.java)** - Bildverarbeitung mit Arrays
- **[VerbrauchBenzin.java](src/Verzweigungen_2/VerbrauchBenzin.java)** - Benzinverbrauchsberechnung

**Fokus:** if-else Verzweigungen, Boolean-Logik, Methoden, Arrays

## Zweck
Mitschriften und praktische Übungen zum Erlernen der Java-Programmiergrundlagen im Schulkontext.
